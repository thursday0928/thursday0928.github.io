可能是全网最好用的mac安卓MySQL-Python指南

发表于 2021-09-18 分类于 工具

在mac下用pip安装MySQL-Python的时候，执行pip install MySQL-python，有时候会遇到如下报错(通常也是最常见的)：

网络上一般提供的方法有：
1.执行export PATH=$PATH:/usr/local/mysql/bin,指定一下路径。
结论：行不通，殊不知新版的mysql安装路径与这个根本不一样。
2.执行brew install mysql-connector-c
然后再执行pip install mysql-python
结论：行不通+1
3.（神奇的操作来了）找到自己的mysql安装目录，在其他的某个目录下找到一个叫mysql.h的文件，将其改成my_config.h。。。。。
结论：行不通+1，会报找不到mysql.h。。。。

以上办法经过证明都是不可行的。终极办法来了：
去网上下一个my_config.h:
curl https://raw.githubusercontent.com/jacobjohn2016/MySQL-Medium/main/my_config.h -o my_config.h
再将该文件放在mysql文件夹下 /usr/local/Cellar/mysql/8.0.21_1/include/mysql
然后再执行pip install MySQL-python，此时我们就发现能安装上了！！！
